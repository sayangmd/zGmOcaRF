```markdown
# 项目简介

## 核心功能
- 系统首页
- 系统用户管理
- 社团类型管理
- 社团信息管理
- 社团成员管理
- 社团活动管理
- 通知信息管理
- 入团申请记录
- 费用申请记录

## 适用场景
- 校园社团组织内部管理
- 社团活动组织与协调
- 会员信息维护与入团申请
- 费用申请与记录
- 通知发布与接收

## 技术特点
- 采用Springboot框架构建后端，确保系统的稳定性和扩展性
- 使用Mybatisplus简化数据库操作，提高开发效率
- 前端使用Vue和ElementUI，提供用户友好的界面体验
- 运行环境包括JDK1.8、NodeJS、Maven和MySQL8，确保系统的兼容性和可维护性
- 系统管理员和社团团长的权限管理功能，确保社团管理的安全性
- 支持远程部署，方便用户在不同环境下使用
```
